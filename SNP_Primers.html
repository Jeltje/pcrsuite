SNP Primers - Input form for Primer3

Back to index

This is an input form for creating primers around SNPs in genomic DNA. The program will design primers around all the SNPs, except for the ones of which the position is 'ambiguous'. For primer design, the Primer3 program is used. You can change the default settings below. The input file is a GenBank sequence.
