How to obtain a GenBank file

Note: The first image in this example is outdated, but the rest of the instructions should still work.

If you want to download a GenBank formatted file that contains your gene, the best (and possibly the only?) way to do it is with NCBI's MapViewer
Below is a simple way to do it in four steps


1. Select the organism and fill in the gene name (Note: this image is outdated. You can select organism on the left hand side of the MapViewer page and fill in the gene below that.)
2. Next, select the Genes_seq link
3. Select Download/View Sequence/Evidence link
4. Select GenBank and Save to Disk

That's all. You are now ready to create Genomic_Primers
